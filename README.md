# CRATES 
## Routes image

The routes image was created using GIMP 2.10. XCF file dimensions are 1795px x 1500px

### Adding a route

Open /assets/images/crates/willship-crates-routes.xcf

Create a new layer for the route. Use the path tool to create a new start and end point, add curves. Stroke the path with 7px #EF4123.

Copy and paste a new arrow from the arrow layer, rotate and move it as appropriate.

City text is 30px Sans-Serif #002157. Add the new city name to the cities layer and move as appropriate.

Export as a PNG resize if you want to, and upload.
